Die Komponente

Die Klasse

                     
                        import {Component} from "@angular/core";

                        @Component({
                            selector: 'bv-app',
                            styleUrls: ['./app.component.scss']
                            templateUrl: './app.component.html'
                        })
                        export class AppComponent {
                            title = "Hallo Welt";
                        }

@Component

  • muss zwingend vorhanden sein
  • statt einer templateUrl kann auch ein Template Inline mit Backtips ` verwendet werden
                     
                        @Component({
                            selector: 'bv-app',
                            styles: [`.h1 {
                                font-size: 1.2 rem;
                            }`]
                            template: `
Ich bin ein Titel
`
                        })
                        export class AppComponent {}

Aufgabe 1

  • Baue die bestehende Komponente auf Inline um
  • Nutze ein inline Template mit einer H1-Überschrift
  • Style die Überschrift in einer schönen Farbe #FF00FF
  • Denke an die Backticks und die Propertynamen

Mögliche Lösung

                     
                        import {Component} from '@angular/core';

                        @Component({
                            selector: 'app-root',
                            template: `
Schöne Farbe
`,
                            styles: [`h1 {color: #FF00FF;}`]
                        })
                        export class AppComponent {
                            title = 'app works!';
                        }
